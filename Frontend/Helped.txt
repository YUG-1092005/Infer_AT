ðŸ“„ 2. load-navbar.js â€” JavaScript Loader

This script will fetch the navbar.html file and insert it into a container in your main pages.

// load-navbar.js
document.addEventListener("DOMContentLoaded", function () {
  fetch("navbar.html")
    .then((response) => {
      if (!response.ok) {
        throw new Error("Failed to load navbar: " + response.status);
      }
      return response.text();
    })
    .then((data) => {
      document.getElementById("navbar").innerHTML = data;
    })
    .catch((error) => {
      console.error("Error loading navbar:", error);
    });
});

ðŸ“„ 3. index.html (and any other page)

In each HTML page, youâ€™ll create a <div id="navbar"></div> placeholder, and load the load-navbar.js script.

<!-- index.html -->
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8" />
  <meta name="viewport" content="width=device-width, initial-scale=1.0" />
  <title>Home</title>
  <link rel="stylesheet" href="css/style.css" />
</head>
<body>

  <!-- Navbar placeholder -->
  <div id="navbar"></div>

  <!-- Your page content -->
  <main>
    <h1>Welcome to the Home Page</h1>
  </main>

  <!-- Load the navbar via JavaScript -->
  <script src="load-navbar.js"></script>
</body>
</html>


Do the same in about.html, contact.html, etc.
